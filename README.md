\# Real-Time Fraud Detection System

## Dataset
Download from:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

\## Overview

This project implements a real-time fraud detection system using Kafka and Machine Learning.



\## Architecture

Producer → Kafka → Consumer → ML Model → Dashboard



\## Features

\- Real-time transaction streaming using Kafka

\- Fraud detection using Logistic Regression

\- Persistent logging of results

\- Live dashboard using Streamlit



\## Technologies

\- Python

\- Apache Kafka (Docker)

\- Scikit-learn

\- Streamlit

\- Pandas



\## How to Run

1\. Start Kafka using Docker

2\. Run consumer.py

3\. Run app.py (dashboard)

4\. Run producer.py



\## Output

\- Real-time fraud detection

\- Dashboard visualization
